# SuperApp

**Суперапп** — это кроссплатформенное приложение на языке Java для операционной системы Linux, 
предназначенное для управления файловой системой, мониторинга системных процессов и взаимодействия с ОС через встроенный терминал.

## Описание

Приложение предоставляет единый интерфейс для работы с файлами, съёмными устройствами и системными утилитами. Интерфейс реализован на библиотеке **Java Swing**.

### Основные возможности
- **Управление файлами:** Создание, удаление, копирование, перемещение, переименование файлов и папок.
- **Работа с накопителями:** Автоматическое обнаружение и отображение съёмных устройств (USB) в дереве файлов.
- **Встроенный терминал:** Выполнение команд Linux непосредственно из интерфейса.
- **Системные утилиты:** Запуск монитора ресурсов, управления дисками, настроек сети.
- **Drag-and-Drop:** Перетаскивание файлов между приложением и системой.
- **Мониторинг процессов:** Отслеживание запущенных процессов, формирование отчетов.
- **Защита данных:** Ограничение доступа к системной папке `System`.
- **Межпроцессное взаимодействие:** Всплывающие окна с информацией о системе через очередь сообщений.

## Требования

Для запуска приложения необходимо наличие следующего ПО:
- **ОС:** Linux (рекомендуется Ubuntu).
- **Java:** JRE или JDK версии **17** или выше.
- **Графическая среда:** Поддержка Swing/AWT.

## Установка и запуск

1. Клонируйте репозиторий или скопируйте исходный код проекта.
2. Скомпилируйте проект.
3. Запустите главный класс `Main`.

## Использование
### Главное меню
При запуске открывается главное окно с деревом файлов. Доступны следующие меню:
- Файл: Сохранить отчет, удалить, создать (файл/папку), выход.
- Правка: Копировать, вставить, копировать в...
- Системные утилиты: Терминал, мониторинг дисков, настройка сети, управление процессами.
- Справка: Информация о программе, горячие клавиши.
- Поиск: Поиск файлов и папок во вложенных директориях.
- Доп. окно: Запуск всплывающего окна с системной информацией.
### Операции с файлами
- Создание: ПКМ по папке → Создать → Файл или Папку.
- Удаление: Выделить файл → нажать DELETE или ПКМ → Удалить (файл перемещается в корзину).
- Очистка корзины: ПКМ по папке Корзина → Очистить корзину.
- Копирование/Перемещение:
  - Через буфер обмена (ПКМ → Копировать/Вставить).
  - Drag-and-Drop (перетаскивание мышью).
- Копирование в конкретную директорию (ПКМ → Копировать в...).
- Переименование: ПКМ по файлу/папке → Переименовать.
Папка System защищена. Попытки удаления, перемещения или изменения файлов в ней блокируются сообщением об ошибке.
### Работа с съёмными устройствами
При подключении USB-накопителя программа автоматически сканирует директорию /media. Устройство отображается как отдельный узел в дереве файлов.
При изъятии устройства папка может оставаться в интерфейсе до перезапуска приложения.
### Встроенный терминал
Открывается через меню Системные утилиты → Терминал или горячую клавишу Shift + T.
Поддерживаемые команды:
- ps — список процессов.
- kill <PID> — остановить процесс.
- killall <имя> — остановить все процессы с именем.
- lsusb / lspci — информация об устройствах.
- uptime — время работы системы.
- nice / renice — управление приоритетами.
- pidof <имя> — найти PID процесса.
- help — справка.
- sth <имя> — сохранить историю терминала в файл.
### Отчетность
Для сохранения отчета о запущенных процессах:
1. Меню Файл → Сохранить отчёт.
2. Введите имя файла.
3. Отчет сохранится в папку Документы.
### Горячие клавиши
| Клавиши | Действие |
| --- | --- |
| DELETE | Переместить в корзину |
| Shift + C | Копировать файл |
| Shift + V | Вставить файл |
| Shift + E | Завершить работу приложения |
| Shift + A | Копировать папку |
| Shift + F | Поиск файлов |
| Shift + T | Открыть терминал |
| Shift + D | Запуск всплывающего окна |
